Methodology

Fund Index = (Current Portfolio Value / Base Portfolio Value) × 100
Base Portfolio Value = Σ(Units × Actual Purchase Price) = Total Cost Basis
Current Portfolio Value = Σ(Units × Current Stock Price)
Benchmark Index = (Current ETF Price / ETF Price on Base Date) × 100

Benchmark prices are Adjusted Close (accounts for dividends and splits).
Fund and Benchmark are both rebased to 100 at the first snapshot date for direct comparison.
